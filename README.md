## 下载安装  
- IDEA-> File -> Settings -> Plugins -> Browse Repositories,输入mybatis lite，点install安装即可
- 点击 [这里](https://plugins.jetbrains.com/plugin/10921-mybatis-lite) 下载最新版进行离线安装，IDEA-> File -> Settings -> Plugins -> Install plugin from disk
